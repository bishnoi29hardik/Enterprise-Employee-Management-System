# Enterprise Employee Management System (EEMS)

A comprehensive **Employee Management System** developed using **MySQL 8** to manage employee records, departments, job roles, attendance, and performance reviews.

The project demonstrates relational database design, normalization, SQL query writing, reporting, and database optimization techniques commonly used in enterprise applications.

---

## 🚀 Features

- Employee Management
- Department Management
- Job Role Management
- Employee Role History
- Attendance Tracking
- Performance Reviews
- Normalized Database Design (3NF)
- Primary & Foreign Keys
- Constraints (NOT NULL, UNIQUE, CHECK)
- Views for Reporting
- Indexes for Performance Optimization
- Analytical SQL Queries
